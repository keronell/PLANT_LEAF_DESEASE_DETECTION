================================================================================
                        DATASET OVERVIEW
================================================================================

Main Dataset (Plant_leaf_diseases_dataset_with_augmentation):
  - Source: PlantVillage dataset with augmentation
  - Total Classes: 39
  - Total Images: ~61,486
  - Format: Plant___Disease or Plant___healthy
  - Purpose: Primary training dataset

Plant_doc Dataset:
  - Source: Secondary validation/test dataset
  - Total Classes: 30
  - Total Images: ~2,000+
  - Format: Plant_disease_leaf or Plant_leaf
  - Purpose: Validation and testing

FieldPlant Dataset:
  - Source: Real-world field images with bounding box annotations
  - Total Classes: 27
  - Total Images: 8,580 (after cropping from bounding boxes)
  - Format: Plant___Disease or Plant___healthy (after processing)
  - Purpose: Domain adaptation testing, real-world field conditions

================================================================================
                              NOTES
================================================================================

1. Class Naming Conventions:
   - Main Dataset: Uses "Plant___Disease" format (e.g., "Tomato___Late_blight")
   - Plant_doc: Uses "Plant_disease_leaf" format (e.g., "Tomato_leaf_late_blight")
   - FieldPlant: After processing, matches Main Dataset format

2. Class Overlaps:
   - Some diseases appear in multiple datasets with different naming
   - Example: "Corn___leaf_blight" (Main/FieldPlant) vs "Corn_leaf_blight" (Plant_doc)

3. Unique Classes:
   - FieldPlant contains unique classes not in other datasets
   - Especially for Corn (15 unique classes) and Cassava (5 classes)

4. Healthy Classes:
   - Most plants have a "healthy" class in at least one dataset
   - Format varies: "Plant___healthy" vs "Plant_leaf"

5. Most Represented Plants:
   - Tomato: 25 total classes (most diverse)
   - Corn: 22 total classes (especially in FieldPlant)
   - Apple: 7 total classes
